## Installation
Github:

    git clone https://github.com/I2rys/DNitroGen
    
NPM Packages:
```
npm i discordnitro
```

## Usage
```
node index.js <amount> <output>
```

+ amount - The amount of Discord nitros link to generate.
+ output - The output of the results.
